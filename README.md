# TokenScope — Real-Time LLM Token Budget Manager

Stop missing budget blowouts and start tracking your LLM token spend in real-time.

## Features
- **Real-Time Visibility:** Track usage across all your AI providers.
- **Proactive Alerts:** Receive Slack or Discord notifications at 50/80/95% of your token budget.
- **Per-Project Allocation:** Assign token quotas to individual projects and teams.
- **Multi-Provider Support:** Works with OpenAI, Anthropic, Gemini, and more.

## Quick Start
```bash
npm i -g tokenscope
tokenscope login
tokenscope track --provider openai --project my-project
```
